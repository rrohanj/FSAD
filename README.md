# VibeCheck: Sustainable Fan-Led Logistics Protocol 🌱

VibeCheck is a decentralized demand-verification engine designed to help artists tour sustainably. By proving audience density before booking, we eliminate financial risk and reduce carbon footprints.

## 🚀 Features

- **Dynamic Artist Proposals:** Propose new nodes (artists/cities) to the network.
- **Demand Tracking:** Real-time pledge system with visual progress bars.
- **Analytics Dashboard:** Geographical demand visualization across cities.
- **High-Fidelity UI:** Premium dark/light mode with synchronized logistics tickers.

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS3 (Custom Modules), Lucide React.
- **Backend:** Spring Boot, Java, Maven.
- **Database:** RESTful API Integration.

## 📦 Installation

1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/VibeCheck-FullStack.git`
2. Start Backend: `cd vibecheck-backend && mvn spring-boot:run`
3. Start UI: `cd fan-crowdsourcer-ui && npm start`
